Mapty Workout Tracker

Mapty is a workout tracker built with HTML, CSS, and JavaScript. It uses geolocation and the Leaflet map library to log running and cycling workouts on a live map.

What this app does

  • Finds your current location and centers the map automatically.
  • Displays a "You Are Here" marker for the current position.
  • Allows adding workouts by clicking on the map.
  • Supports running and cycling entries, each with type-specific inputs.
  • Saves workout data in localStorage so it remains after reload.
  • Allows clicking a saved workout to move the map to that workout.

How to use

  1. Open the page and allow location access.
  2. Close this modal using the × button or by clicking outside it.
  3. Click anywhere on the map to add a workout.
  4. Select `Running` or `Cycling`.
  5. Enter `Distance`, `Duration`, and `Cadence` or `Elevation`.
  6. Submit the widget to create the workout entry.
  7. Click the workout item in the sidebar to fly to the saved marker.

Class structure

  • Workout — base class for all workouts with `date`, `id`, `coords`, `distance`, `duration`, and `description`.
  • Running — extends `Workout`, adds `cadence`, calculates `pace`, and sets `type = 'running'`.
  • Cycling — extends `Workout`, adds `elevation`, calculates `speed`, and sets `type = 'cycling'`.
  • App — main controller that loads the map, handles events, renders workouts, and syncs localStorage.

Files in this project

  • index.html — page structure, workout form, modal, and map container.
  • style.css — layout, modal styling, sidebar and workout card design.
  • script.js — map behavior, workout classes, form handling, storage, and modal control.

Technical notes

Workouts are stored in localStorage as JSON. The app restores saved workout data on reload but uses stored values directly rather than re-instantiating class methods from JSON.

The modal is intentionally shown on every load to provide instructions and project details.
